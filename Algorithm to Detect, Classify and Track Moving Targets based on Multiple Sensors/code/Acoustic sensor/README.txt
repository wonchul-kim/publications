
-------------------------------------------------------
-------------- Saving acoustic data ---------------
-------------------------------------------------------

- 1. Register 'NI cDAQ 9191' using 'NI Measurement & Automation Explorer'
- 2. Open 'getAcousticData.vi'
- 3. Set connected microphone parameters by double clicking 'DAQ assistant' node
- 4. Implement the vi
- 5. Type file name in the pop-up window (ex. 'test.csv')
- 6. Press stop button.
